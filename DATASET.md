# Dataset

**Source:** Instacart Market Basket Analysis  
**Link:** https://www.kaggle.com/datasets/psparks/instacart-market-basket-analysis

## Automatic Download
Data downloads automatically on first notebook run via `kagglehub`.

## Manual Download (Optional)
1. Download from Kaggle link above
2. Place in `data/raw/` directory
3. Update path in `src/data_utils.py` if needed

## Dataset Details
- **Users:** 206,209
- **Products:** 49,688
- **Orders:** 3.3M
- **Time period:** Training data through temporal split
